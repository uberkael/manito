# Manito
MCP for querying and exploring Unix man pages


## Install

```json
"manito": {
	"command": "npx",
	"args": [
		"-y",
		"tsx",
		"path_to/main.ts"
	]
}
```

## Debug

Ejecutar shttps.ts
`npx -y @modelcontextprotocol/inspector `
conectar con el token y [http://localhost:3000/mcp](http://localhost:3000/mcp)
